# 导师相处详细策略

蒸馏自第5章「与导师的相处」、第6章「选错了导师怎么换」、第8章「识破导师的push和pua」、第9章「导师鸿沟之这个好发论文」

## 免责声明

如果你的导师不push你、尊重你想法、帮你科研，**以下所有方法不适用**。你就该怎样怎样，适当帮导师做点举手之劳也是应该的。

## 第一层：示弱

### 为什么示弱
导师自己也是「大牛马」——他有评职称压力、业绩考核压力、经济压力。你释放全部实力＝变相鼓励他给你加码。

### 如何示弱
1. **给自己定时间配额**：个人事务Xh + 科研Yh + 导师分配的任务Zh
2. **超过配额就停**，加班熬夜通通不可行
3. **导师说的deadline不是真deadline**——摸清他的真实底线
4. 口头禅：「我能力差、我不行、我会努力」

### 副作用
遇到奇葩导师会骂你傻逼垃圾。解决方案：练厚脸皮。这是职场生存法则，迟早要会的。

## 第二层：交好

### 为什么交好
导师再烂也是你导师，未来很多事需要他签字。

### 如何交好
1. 力所能及的事能帮则帮
2. 耗时不过分的事能做则做
3. 累活难活 → 公开场合先接受，私底下找理由推
4. 言语上：好话不断，瞅准机会夸「老师这个想法不错」
5. 生活上：私人订制节日祝福 + 送土特产（不贵重，人情往来）
6. **明修栈道暗度陈仓**：明面上答应+装努力，私下做自己的事

## 第三层：沟通

### 和导师沟通
90%的矛盾来自信息不匹配。导师默认你是想搞科研的好孩子。如果你不想，**尽早说清楚**。

- 想做考公 → 直接说想做考公
- 想水论文 → 直接说只想毕业
- 学得慢 → 就说学得慢
- 不要让导师对你产生过高期待

### 和辅导员沟通
问往届师兄姐毕业情况、学院毕业要求、评奖评优细则。不要被互联网洗脑，也不要觉得自己是例外。

### 和师兄姐沟通
尽量有一两个聊得来的师兄姐。信息差很重要。实在合不来也别勉强。

## 第四层：莫较真

**正确与否不重要，听话很重要。**

- 导师说一个荒唐的想法 → 不要马上反驳
- 这件事要你做 → 去尝试一下，真做不出来再解释
- 逼你做不可能的事 → 给他搞个「结果」来——他是第一负责人他都不怕你怕啥
- 他异想天开 → 你陪着他异想天开
- 他开心了，你自然就毕业了

大多数研究生不能毕业不是因为学术能力，而是花了太多时间处理和导师的关系。

## 第五层：斗争（底线策略）

**前置条件：你已完成学院培养计划的毕业要求。**

如果导师仍然极度过分（辱骂、人身攻击、让你做家务等）：
1. 搜集证据（聊天记录、录音、邮件）
2. 找辅导员
3. 找学院领导
4. 找学校
5. 找教育厅/教育部
6. 报警

**⚠️ 只有在你情绪快绷不住、实在没有办法的时候才走这条路。**
舆论天然站在「他是导师你是学生」这一边。

### 换导师

**互选前（录取后~开学前）：**
- 如果发现导师不匹配，直接溜
- 越早溜越好，开学后别人早忘了你是谁

**互选后（开学后）：**
- 非必要不换（流程麻烦，伤和气）
- 真的受不了 → 先以德服人 + 装可怜 → 最终找学院协调

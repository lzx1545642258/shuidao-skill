# 三年时间规划表

蒸馏自第17章「三年的时间规划」

## 标准时间线（每天4-6小时有效科研时间）

```
研一（入学年）
├── 暑假（录取后~9月）
│   ├── 未找导师 → 先看完指南再找，享受生活为主
│   ├── 已找导师 → 略看方向相关基础，享受生活为主
│   └── ⚠️ 这是人生最后一段无忧无虑的时光，以享受为主
│
├── 研一上学期（9月~1月）
│   ├── 打基础（配合暑假的懒散学习，开学后花半年）
│   ├── 定毕业目标（考公/进厂/读博）
│   ├── 定研究方向
│   └── 开始看综述
│
└── 研一下学期（2月~6月）
    ├── 继续打基础
    └── 开始找基准模型

研二（核心产出年）
├── 研一暑假（7月~8月）
│   └── 基准模型应该已经跑通了
│
├── 研二上学期（9月~1月）
│   ├── 找模块＋精读几篇论文
│   ├── 基准模型可正常运行 + 有多个模块候选
│   └── ⚠️ 想进厂的同学：此时应**完成全部科研任务**
│
├── 研二寒假（2月）
│   └── 缝模块、做实验
│
├── 研二下学期（3月~6月）
│   ├── 写论文（1-2个月）
│   ├── 投稿
│   └── ⚠️ 进厂的同学 → 此时应已开始**实习**
│
└── 研二暑假（7月~8月）
    ├── 大论文初步构思
    └── 考公的同学准备国考

研三（毕业年）
├── 研三上学期（9月~1月）
│   ├── ⚠️ 9-10月：国企/央企校招（考公的同学也值得投）
│   ├── 10月后：大论文动笔
│   ├── 国考（11月/12月）
│   └── 秋招
│
├── 研三寒假
│   └── 大论文写作
│
└── 研三下学期（2月~6月）
    ├── 大论文完成（3月/4月）
    ├── 盲审
    ├── ⚠️ 考公考编黄金时间（省考、各地编制招聘）
    ├── 答辩（5月/6月）
    └── 毕业🎉
```

## 不同目标的时间安排差异

### 进厂方向
| 阶段 | 调整 |
|------|------|
| 科研 | 每天6-8小时有效时间（007节奏），**研二上完成全部科研** |
| 实习 | **研二下开始实习**，争取半年以上 |
| 论文 | 研二上就投稿，不拖延 |
| 原因 | 大厂最看重实习经历，HR看不懂你的论文 |

### 考公考编方向
| 阶段 | 调整 |
|------|------|
| 科研 | 正常进度即可 |
| 国考 | 如果想稳国考，参考进厂同学的压缩进度 |
| 黄金期 | 研三下学期（省考、各地编制招聘密集期） |
| 论文 | 研三上动笔即可 |

### 读博方向
| 阶段 | 调整 |
|------|------|
| 科研 | 高质量论文优先（SCI二区以上） |
| 论文 | 必须发高质量论文，数量不重要 |
| 额外 | 经营导师关系（需要引荐） |

## 关键提醒

- **如果发现自己慢了 → 压缩休息时间赶进度，别积压到研三**
- **如果发现自己快了 → 可以停下来犒劳自己，也可以继续冲**
- 考公的同学：考公≠不写论文，先满足毕业要求再准备考试
- 进厂的同学：没有实习经历想直接毕业进厂 ≈ 白送
- 每天有效时间≠坐在工位的时间，扣除刷手机、发呆、聊天才是有效时间
